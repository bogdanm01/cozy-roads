# Cozy Roads

A warm, low-poly, endless-driving game prototype built with Godot 4.

## First playable

The current milestone is a small driving toy used to tune the feel before procedural world generation begins.

### Controls

- `W` / `Up`: accelerate
- `S` / `Down`: brake and reverse
- `A D` / arrow keys: steer
- `R`: reset the car
- Controller: left stick plus triggers

## Run

Open `project.godot` with Godot 4.7 or run:

```sh
/Applications/Godot.app/Contents/MacOS/Godot --path .
```
